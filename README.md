vim-monit
=========

Description
-----------

[Vim](http://www.vim.org/) plugin for [Monit](http://mmonit.com/monit/).

This plugin provides:

  * Syntax highlight
  * Snipmate snippets


Installation
------------

Use [Vundle](https://github.com/gmarik/vundle), to install this plugin:

    # inside vim
    :BundleInstall 'mv/mv-vim-monit'

    # Add to ~/.vimrc
    Bundle 'mv/mv-vim-monit'
